# RefinPick 프로젝트 개요

## 1. 프로젝트 개요

**프로젝트명**  
RefinPick

**한줄 설명**  
사용자가 현재 대출 조건을 입력하면 월 상환액, 총 이자, 총 상환액을 계산하고, 다른 금리·기간 상품으로 갈아탔을 때의 절감 효과를 비교해주는 금융 의사결정 지원 서비스

**목적**
- 대출 상환 구조를 직관적으로 이해할 수 있도록 돕기
- 금리, 기간, 상환방식 변화가 실제 부담에 어떤 영향을 주는지 보여주기
- 대출 갈아타기(대환)의 경제적 효과를 정량적으로 비교하기
- 고정금리와 변동금리의 차이를 사용자 교육형으로 설명하기

## 2. 핵심 기능

### 1) 현재 대출 상환 계산
- 월 상환액
- 총 이자
- 총 상환액
- 남은 상환 부담 계산

### 2) 갈아타기 상품 비교
- 기존 대출과 신규 대출의 월 상환액 비교
- 총 이자 차이 비교
- 총 상환액 차이 비교
- 예상 절감액 계산

### 3) 상환방식별 비교
- 원리금균등상환
- 원금균등상환
- 만기일시상환

### 4) 고정금리 vs 변동금리 이해 기능
- 금리 유지 시
- 금리 1%p 상승 시
- 금리 1%p 하락 시

### 5) 시각화 기능
- 월별 상환 스케줄 표
- 원금/이자 비중 변화 그래프
- 기존 대출 vs 갈아타기 상품 비교 차트
- 누적 상환액 비교 그래프

## 3. 사용자 시나리오

### 시나리오 1. 현재 대출 부담 확인
1. 사용자가 현재 대출 원금, 금리, 기간, 상환방식을 입력
2. 시스템이 월 상환액과 총 이자를 계산
3. 사용자는 자신의 대출 구조를 이해

### 시나리오 2. 대출 갈아타기 검토
1. 사용자가 현재 대출 조건을 입력
2. 신규 대출 상품 금리와 기간을 입력
3. 시스템이 두 조건을 비교
4. 사용자는 월 부담 감소액, 총 절감 이자를 확인
5. 갈아타기 여부를 판단

### 시나리오 3. 금리 변동 리스크 체험
1. 사용자가 변동금리 상품을 선택
2. 금리 상승/하락 시나리오를 적용
3. 시스템이 월 상환액 변화와 총 이자 증가폭을 보여줌
4. 사용자는 변동금리 리스크를 직관적으로 이해

### 시나리오 4. 상환방식 선택 비교
1. 사용자가 같은 대출금액과 금리로 상환방식을 바꿔봄
2. 시스템이 방식별 결과를 나란히 표시
3. 사용자는 자신의 상황에 맞는 방식을 선택

## 4. 필요한 화면 구성

### 1) 메인/랜딩 화면
- 서비스 소개
- 내 대출 계산하기 시작 버튼
- 주요 기능 요약
- 고정금리/변동금리 설명 카드

### 2) 현재 대출 입력 화면
- 대출 원금
- 연이율
- 대출 기간
- 상환방식
- 대출 실행일 또는 잔여 기간
- 거치기간 여부

### 3) 현재 대출 결과 화면
- 월 상환액
- 총 이자
- 총 상환액
- 월별 상환 스케줄
- 원금/이자 비중 그래프

### 4) 갈아타기 비교 화면
- 신규 금리
- 신규 기간
- 신규 상환방식
- 중도상환수수료
- 부대비용
- 기존 대출 vs 신규 대출 비교표
- 총 절감액
- 손익분기 시점

### 5) 금리 시나리오 화면
- 고정금리 vs 변동금리 탭
- 금리 변동 시나리오 비교
- 결과 비교 차트

### 6) 요약 리포트 화면
- 입력 요약
- 추천 해석
- 갈아타기 유불리 인사이트

## 5. 데이터 항목

### 기본 입력 데이터
- 대출금액
- 연이자율
- 대출 기간
- 상환방식
- 금리 유형
- 잔여 원금
- 잔여 기간
- 대출 시작일

### 비교용 데이터
- 신규 대출 금리
- 신규 대출 기간
- 신규 상환방식
- 중도상환수수료
- 인지세/부대비용
- 우대금리 여부

### 계산 결과 데이터
- 월 상환액
- 총 이자
- 총 상환액
- 월별 원금 상환액
- 월별 이자 상환액
- 누적 상환액
- 갈아타기 후 절감액
- 비용 반영 후 순절감액

- 기준금리 변화폭

- 최대 금리 상한
- 시나리오별 예상 상환액

## 6. 개발 우선순위

### 1순위: 핵심 계산 기능
- 대출 입력 폼
- 월 상환액 계산
- 총 이자/총 상환액 계산
- 상환방식별 계산 로직

### 2순위: 갈아타기 비교 기능
- 기존 대출 vs 신규 대출 비교
- 절감액 계산
- 중도상환수수료 반영

### 3순위: 결과 시각화
- 상환 스케줄 표
- 그래프/차트
- 비교 UI

### 4순위: 금리 시나리오 기능
- 고정 vs 변동 비교
- 금리 상승/하락 시뮬레이션

### 5순위: 해석/추천 기능
- 계산 결과 해석 문구
- 사용자 친화적 금융 교육 요소 추가

## 7. 포트폴리오용 차별점

### 1) 의사결정 지원 서비스
단순 계산이 아니라 갈아타기 판단을 지원하는 서비스 구조

### 2) 금융이론 + 실사용성 결합
경제금융학부 전공 역량과 실제 서비스 기획을 함께 보여줌

### 3) 교육형 UX
고정금리와 변동금리 차이를 사용자가 직접 체험 가능

### 4) 실제 비용 반영
중도상환수수료와 부대비용까지 반영한 순절감액 제공

### 5) 시각화 중심 설계
상환 구조를 표와 그래프로 직관적으로 전달

### 6) 확장 가능성
실제 금융상품 API, 신용점수, DSR 반영 등으로 확장 가능
