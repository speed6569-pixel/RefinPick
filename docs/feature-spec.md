# RefinPick 기능 명세

## 핵심 모듈

### 1. 대출 입력 모듈
- 현재 대출 정보 입력
- 신규 대출 조건 입력
- 금리 유형 선택
- 상환방식 선택

### 2. 계산 엔진
- 원리금균등상환 계산
- 원금균등상환 계산
- 만기일시상환 계산
- 총 이자, 총 상환액 산출
- 절감액 계산
- 순절감액 계산

### 3. 비교 분석 모듈
- 기존 대출과 갈아타기 상품 비교
- 월 상환액 변화량 계산
- 총 부담 변화량 계산
- 손익분기 분석

### 4. 시나리오 모듈
- 고정금리 시뮬레이션
- 변동금리 상승/하락 시나리오
- 시나리오별 비교 결과 제공

### 5. 리포트 모듈
- 핵심 수치 요약
- 추천 메시지 생성
- 포트폴리오 발표용 요약 뷰 제공
