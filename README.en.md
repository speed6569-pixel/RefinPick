# RefinPick

A loan comparison, refinancing, and repayment simulator built as a portfolio project for an economics and finance student.

## Overview

RefinPick helps users calculate monthly payments, total interest, and total repayment from their current loan details, then compare refinancing options under different rates and terms to estimate savings.

## Core value
- Understand current repayment burden
- Compare refinancing benefits clearly
- Explore repayment-method differences
- Learn fixed-rate vs variable-rate risk through scenarios

## Stack
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Run
```bash
npm install
npm run dev
```
